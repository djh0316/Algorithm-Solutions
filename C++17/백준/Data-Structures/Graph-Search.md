# 🔍 그래프 탐색 (Graph Search)

### 1. BFS (Breadth-First Search, 너비 우선 탐색)
- **특징**: 시작 노드에서 가까운 노드부터 탐색.
- **주로 사용되는 경우**: 최단 경로 찾기.
- **자료구조**: `Queue` 활용.

### 2. DFS (Depth-First Search, 깊이 우선 탐색)
- **특징**: 한 방향으로 갈 수 있을 때까지 깊게 탐색.
- **주로 사용되는 경우**: 모든 노드 방문, 경로의 특징을 저장해야 할 때.
- **자료구조**: `Stack` 또는 `재귀 함수` 활용.
